# План тестирования

## Цель
Проверить, что библиотека работает правильно.


## Что проверяем

- создание меню
- добавление пунктов
- вывод меню
- выбор пункта
- выполнение действия
- ввод числа
- ввод строки


## Как проверяем

- через Unit-тесты (assert)
- через запуск программы вручную


## Результат

Тесты считаются успешными, если:

- программа не падает
- меню работает
- команды выполняются
- ввод работает без ошибок
